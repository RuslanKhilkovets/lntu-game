# LNTU PIXEL DODGE

## Опис

LNTU PIXEL DODGE — це проста аркадна гра, де гравець керує прямокутником у нижній частині екрану, уникаючи ворогів, що падають згори. Гра написана на HTML, CSS та JavaScript із використанням canvas для рендеру графіки. Проєкт містить кнопки для перезапуску гри та відкриття туторіалу.

## Функціонал

- Керування гравцем за допомогою стрілок вліво/вправо
- Вороги падають згори, гравець має їх уникати
- За кожного пропущеного ворога нараховується бал
- При зіткненні з ворогом зменшується кількість життів
- Гра завершується, коли життя = 0
- Кнопка "Restart Game" для перезапуску
- Кнопка "Як грати?" відкриває модальне вікно з інструкцією
- Піксельний стиль оформлення

## Запуск

1. Відкрийте `index.html` у браузері.
2. Для запуску юніт-тестів виконайте у терміналі:
   ```
   node js/test.js
   ```
